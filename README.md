# 💰 ExpanseTracker

A Full Stack Expense Tracking Application built using **Spring Boot** and **React.js**.  
This project helps users track their daily expenses, categorize spending, and manage financial records efficiently.

---

## 🚀 Features

- Add new expenses
- View all expenses
- Categorize expenses
- REST API integration
- Full stack architecture (Spring Boot + React)
- Clean component-based UI
- Backend API with Spring Boot
- MySQL database integration
